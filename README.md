# Silent Storm Port

Open-source modernization of the 2003 Nival Interactive game Silent Storm,
based on the 2026 source release. See `../docs/superpowers/specs/` for design.

**Status:** Phase 1 + 1.5 COMPLETE. silent_storm.exe boots into Nival's intermission screen rendered via bgfx D3D11 + 8 shader archetypes + custom font atlas. Verified at 1920×1080, 2560×1440, 3840×2160. Next: Phase 2 audio (FMOD → miniaudio). See `docs/phase-1-completion.md`.
